# Tech.Care Dashboard

A high-fidelity healthcare management dashboard built for visualizing patient medical history and vitals.

## Tech Stack
- **React 19**
- **Vite**
- **Chart.js**
- **Vanilla CSS**

## Prerequisites
- Node.js (v18+)
- npm or yarn

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Development Focus
This project emphasizes **UI Precision**, **Data Accuracy**, and **Premium User Experience**. Every element has been styled to meet professional design standards, ensuring a responsive and highly-readable dashboard for healthcare practitioners.
